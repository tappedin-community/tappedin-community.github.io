"Group Button"

The user will get a list of Group choices based on the groups that he or she is in. If the user is not in any groups the add Groups button won't appear. Once a group has been selected it will appear in a new line in the table and the user can add more groups again and again if he or she desires.

 

"Course Button"

The user will get a list of Course choices based on the courses he or she is enrolled in. If the user is not enrolled in any Courses the course column of the table and the add Course button goes away. Once a course has been selected it will appear in a new line in the table and the user can add more courses again and again if he or she desires.
