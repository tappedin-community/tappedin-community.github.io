Files available within a room

The owner of a room can add files to that room. [1] The list of files that an owner has added is available under the Files link in the left menu bar of each room, as shown below. Files are associated with a room, and can be added every room type (individual rooms, group rooms, and course rooms). Any visitor with access to a room can access the files in that room. [1]

By clicking on Files, a visitor will see a list of Files in that room [1] , each with an (optional) annotation [2] added by the room owner.

Files in of a room can also be "featured." Featured files are highlighted and appear on the Featured Items list on the Welcome page. [2]

If a user clicks a File, it will download to their computer. [1]

Editing Files

By default, each room when created will contain no files. Only the owner of a room can edit the rooms's files. The owner can delete and add new Files to the room. [1]

When the owner of a room clicks on the File link, they see an interface similar to what a vistor would see, except they also see a delete link beside each File [1], as well as two buttons: add file [1] and put checked items in your featured items [2]

To delete a file, the owner just clicks on the delete link. The owner is presented with a confirmation message before the deletion is done, for example:

Are you sure you want to delete the File Lesson Plan 1 Draft?
Yes Cancel

If the user selects Cancel the file is not deleted; if the user selects Yes the file is deleted.

To add a file, the owner clicks the Add File link. She is then presented with a field into which she can type or paste a title [1], an annoation field where she can add comments about the file[2], and an Select File button.

A room owner can add files up to 20 files or 5 megabytes to a room, whichever limit is reached first.[2] When this limit is reached, the upload file button no longer appears. In its place, a message appears that says:

No more than 20 files or 5 megabytes of filespace can exist for a room.

Groups will get 10 megabytes or 50 files, and courses will get 20 megabytes or 100 files.

When the user presses Upload File, a file browser pops up and she can use it to navigate to a file on her hard drive. Any and only files from the users's hard drive can be uploaded.[1]

 

When the user presses Open or Select,, the file is added to her room's Files area, and she returns to that area.[1]

If a newly uploaded file has the same name as an existing file in the list, we could offer to replace the existing file with the new one (might make updating easier) [3]

Finally, each room has a Featured Items list that can be displayed in a sticky note in the rooms Welcome page. Items that can be featured include files, links, and passageways. To add files to the Featured Items list, the owner checks the files she wants to add, and clicks the button put checked items in your featured items.[2]

Notes

The files that a owner adds to a room also automatically appear under their Files under Me, organized by the room that the files are in. To edit a file from Me/Files the owner must go to the room that the file is in and edit it there. [1]

The number of Files that an owner can link to in a given room will be set at 20 files or 5 megabytes maximum [2]. Groups will get 10 megabytes or 50 files, and courses will get 20 megabytes or 100 files.[2]

Hierarchies/folders [3] are not supported in version 1.0.

Rearranging order [3] is not supported in version 1.0. Files are listed in the order that they are added.

Scan uploaded HTML files for javascript that could do malicious things, and not allow upload of such files [2]

Later versions could have a server-side virus checker that prevents uploading of files with viruses [3]
