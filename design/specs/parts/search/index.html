Search

MAIN SEARCH

See searchmain.html

Search of all of TI is a simple text box that appears at the top of all pages. Keyword searching of the entire site is possible from this search text box. Next to this search box, there is a pull down menu where the user can specify which media they wish to search. The default is "All".

POWER SEARCH

See advancedsearch.html

Also next to the search box, there is a link to a Power Search /Advanced Search. The Power Search enables the user to specify more than one media type, subject area, and grade level. These areas for subjects and grade indicators are ways of categorizing the information that the TI staff formalizes and updates based on input from users indicating the keywords they would use to categorize their entries. The system should enable searching of all existing and future media types. Current media types include:

  • Public groups
  • Files
  • Web sites
  • Members
  • Mentors
  • Message Boards
  • Chat logs
  • Help files
  • Currently, courses are not searchable in the Prometheus system since they are not made public. We may want to consider having a listing of courses with the option of faculty listing their member name and email address to be contacted.

The default setting is "All media types". As soon as a user selects a specific media type, the search page reflects that only specified media types will be searched.

KEYWORDS

see Keywords.html

Keyword searches will follow Boolean logic; the default separator between multiple words will be "and". Common Keyword listings will contain TI staff specified categories and keywords. These keywords can be updated based on how users categorize their entries (documents, web sites, profiles, etc.)

RESULTS

See searchresults.html

Results are ordered by percentage of match to search string, with greatest match appearing first. The media type is named next to each entry except if the search is only on one media type. The entry links to the result. Users can easily refine their search from this search results page. "New search" is also context specific, if a context was indicated other than "All" in the original search, it is still marked in the pull down menu. The user can change this setting if they decide to do a new search.
